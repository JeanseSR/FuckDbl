# Fuck DBL - Suppression de doublons par hash

Petit outil personnel pour détecter et supprimer les fichiers doublons (par hash MD5).

## Fonctionnalités
- Analyse récursive d’un dossier
- Filtrage par type de fichiers
- Suppression temporaire (corbeille) ou définitive
- Export du rapport au format CSV

## Utilisation
Lancer `main.py`, sélectionner un dossier, lancer l’analyse, cocher les fichiers à supprimer.

## État
Stable – utilisé quotidiennement par son auteur.
